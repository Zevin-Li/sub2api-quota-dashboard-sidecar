# Changelog

## 0.1.4 - 2026-05-19

### Added

- Added table header sorting for the history view.
- Added filter summary text below the toolbar filters.
- Added explicit collapsed hint text for the recovery main panel.

### Fixed

- Fixed long account names, recovery text, error text, and update-time labels overflowing cards.
- Fixed recovery view collapse state so closing all recovery sub-lanes no longer locks the main recovery panel closed.
- Fixed panel toggle alignment so the arrow container is visually aligned with its title text.
- Fixed outer page background edge mismatch in embedded and scrollable layouts.
- Fixed the refresh button edge bleed on the light hero banner by using a real border and clipped background.

### Changed

- Refined the dashboard color system toward a lighter white-cyan and blue-green palette closer to the default `sub2api` feel.
- Lightened the top hero banner and updated its text, pills, and status card contrast for the new palette.
- Refined card, tab, table, and account accent gradients to reduce warm gold tones in favor of teal and blue accents.
